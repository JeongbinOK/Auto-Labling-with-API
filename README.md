# Auto-Labling-with-API
<br/>

## 설명
---
data에 원하는 이미지를 **1.png ~ n.png** 형식으로 추가하면, 그에 따라서 GPT4 또는 Gemini API를 이용해 Auto Labeling하는 파일입니다.

<br/>

## 모델 리스트
---
### Gemini
- gemini-pro-vision

### GPT4
- gpt-4-turbo-2024-04-09

<br/>
<br/>

## 사용방법
---

### Formatting이 필요한 경우
formatting =True로 설정한다.

<br/>
<br/>

### API 사용할 경우
각각 API_Key와 API_model 명을 각각 설정한다.
